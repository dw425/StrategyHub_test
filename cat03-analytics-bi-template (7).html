KPI vs OKR Framework

üéØ KPI vs OKR ‚Äî When to Use Each
‚úÖ Good KPIs & OKRs
GOOD KPI
"Monthly Active Users (MAU)"
‚úì Specific metric with clear definition
‚úì Measurable from product analytics
‚úì Target: 500K MAU, currently 420K
GOOD OKR
O: "Become the #1 choice for enterprise customers"
KR1: Close 10 Fortune 500 deals (currently 3)
KR2: Achieve SOC2 Type II certification
KR3: Reduce enterprise onboarding time to <2 weeks
GOOD KPI
"Net Revenue Retention (NRR)"
‚úì Formula: (Revenue from existing - churn + expansion) / Starting revenue
‚úì Target: NRR ‚â• 110%
‚úì Shows growth from existing customers
‚ùå Bad KPIs & OKRs
BAD KPI
"Increase customer satisfaction"
‚úó Not measurable ‚Äî what's "satisfaction"?
‚úó No target number

BAD OKR
O: "Ship the new dashboard"
‚úó Objective is a task, not an outcome
‚úó No key results measuring impact
FIX ‚Üí O: "Empower users to find insights faster" KR: "Reduce time-to-insight by 50%"

"Lines of code written"
‚úó Vanity metric ‚Äî doesn't measure value
‚úó Incentivizes wrong behavior

Automates data exploration, identifies trends and anomalies, generates natural language insights, and creates visualizations tailored to your questions.

  • Dashboard generation

analytics_agent.py
# analytics_agent.py - BI Analytics Agent
from crewai import Agent, Task, Crew

analytics_agent = Agent(
    role="Analytics Specialist",
    goal="Transform data into actionable insights",
    backstory="Expert in statistical analysis and visualization",
    tools=[
        SQLQueryBuilder(),
        AnomalyDetector(),
        TrendAnalyzer(),
        ChartGenerator(),
        InsightNarrator(),
    ]
)

analysis_task = Task(
    description="""
    User Question: "Why did revenue drop last month?"
    
    1. Query revenue data for past 6 months
    2. Segment by product, region, customer type
    3. Identify anomalies and contributing factors
    4. Generate visualizations showing breakdown
    5. Write executive summary with recommendations
    """,
    agent=analytics_agent,
    expected_output="Root cause analysis with visualizations"
)

# Execute analysis
crew = Crew(agents=[analytics_agent], tasks=[analysis_task])
result = crew.kickoff()

# Output: "Revenue declined 12% primarily due to..."
